# jarsi.org

Personal site of [jrs8205](https://github.com/jrs8205) — a single static page
showcasing selected open-source projects.

## Stack

- Plain HTML and CSS, no JavaScript, no build step
- Hosted on Cloudflare Workers (static assets)

## Development

Open `public/index.html` directly in a browser, or run a local preview:

```
npx wrangler dev
```

## Deployment

```
npx wrangler deploy
```
